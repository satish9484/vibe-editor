# @vibecode-starters

A collection of minimal, consistent starter templates across popular frameworks. Each starter includes TypeScript, basic linting, and sensible scripts.

Starters:
- nextjs-new: Next.js App Router (TypeScript)
- react-ts: React + Vite (TypeScript)
- vue: Vue 3 + Vite (TypeScript)
- hono-nodejs-starter: Hono on Node (TypeScript)
- express-simple: Express (TypeScript)
- angular: Angular (TypeScript)

Conventions:
- Package manager: npm (you can switch to pnpm/yarn if preferred)
- TypeScript: strict
- Linting: ESLint flat config (where supported)
- Formatting: Prettier

Quickstart (example):
```
cd vibecode-starters/nextjs-new
npm install
npm run dev
```
